# Portfolio d'Екатерина

Ce projet est un site web simple créé pour présenter mes projets personnels et mes compétences en développement web. Il utilise HTML et CSS pour la mise en page et le style.

## Projets inclus
- To-Do App en Python
- Mini-site de recettes en HTML/CSS/JS
- Calculatrice simple en Python

Ce portfolio est destiné à démontrer mes compétences de débutante en programmation et ma motivation pour un apprentissage en alternance.
